India Air Quality Analysis using Python
📌 Overview

This project analyzes real-time air quality data from monitoring stations across India. It focuses on pollution trends, pollutant comparison, hotspot detection, and PM2.5 prediction using Machine Learning.

📊 Dataset
3443 Records
31 States
268 Cities

CO
NH3
NO2
OZONE
PM10
PM2.5
SO2

Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Top Polluted States & Cities Detection

PM2.5 vs PM10 Correlation Analysis
Heatmaps & Visualizations
Linear Regression Prediction Model
